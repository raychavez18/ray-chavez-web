Services

Hello, I'm Ray Chavez and I love making websites. I graduated from the Illinois Institute Of Art-Chicago with a B.A. in Web Design and Interactive Media in 2011. I love developing and finding solutions to complex problems on the web. If I challenge myself everyday while developing websites, I feel that I'm gaining knowledge everyday which is the beauty part of this industry. There is so much to know which makes the internet endless and a beautiful thing.

In addition to pursuing my passion in technology, I love soccer, baseball, mixed martial arts and live music.

Below are the servcies I provide:


Front End Developent

Toolset: HTML5, CSS3, Javascript ES6, jQuery, Vue.Js, Bootstrap, Greensock

This is what drove me to become a web developer. Using the latest technologies to architect and write clean, durable code to build optimized responsive experiences and interactive interfaces across platforms and browsers.

Web Application Development

Toolset: Django, Python, Vue.JS, GIT, AWS, Bitbucket, SourceTree

Whether it's client side or server-side I love creating applications in different languages. Over the years I learned how to build large scale, high-performance, and easy-to-maintain web applications using Django and now learning using modern Javascript frameworks.

Responsive HTML Email Development

Toolset: HTML, CSS, Email on Acid, Litmus

Writing HTML emails can be archaic. I love creating attractive, clean responsive, HTML Emails, that work in over 30 combinations of the most commonly used email clients.

HTML5 Banner Development

Toolset: HTML5, CSS3, Javascript ES6, jQuery, Greensock, Doubleclick

Animation is one of my most favorite things on the planet. With current front-end technologies I develop a master banner from a brief and then quickly and efficiently roll out the master banner in all required formats and sizes. On my free time I love to learn new animations in Javascript as well.

CMS Integration & Migration

Toolset: Wordpress, Drupal and Adobe Experience Manager

Between the most common Content Managment Systems, I have expereince moving and creating new content for large projects with enterprise level clients. Building out landing pages, maintaining plugins and installing a new CMS has been most of my experience.

UI/UX

Toolset: Adobe Photoshop CC, Adobe Illustrator CC, Sketch

Having a design background, I've guided and helped businesses to make rich user interfaces and experiences on the web. I stay up to date with design standards and what's common just as much as I stay up to date with modern web development standards. With my help the development process becomes much easier and provides a great experience on the web.
